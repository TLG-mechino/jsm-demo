Research Data Policy

Overview

JSM encourages transparency and reproducibility in research. Authors are expected to support open science principles by making research data available in accessible and reusable formats wherever possible.

Authors are encouraged to
Deposit research data in trusted repositories
e.g. Zenodo, Figshare, Dryad, or institutional repositories
Provide a Data Availability Statement
Clearly describe where and how the data can be accessed
Cite datasets appropriately
Include DOIs or permanent links when referencing datasets
Share code where possible
Scripts and analysis code improve reproducibility and transparency
Explain any restrictions on data access
State reasons clearly if full data sharing is not possible
Exceptions

Where data cannot be shared due to legal, ethical, or confidentiality reasons, authors should clearly state the justification in the Data Availability Statement. The editorial team will review such cases on an individual basis.
